# SeopMemoApp  

##  프로젝트 개요 

iOS App 개발을 공부하기 위해 진행한 프로젝트로서 사진추가가 가능한 간단한 메모앱입니다.   


## 개발환경   

XCode Version : 11.x
Swift : Swift 5.0 
Target iOS Version : iOS 10.0


## APP 기능

- 메모 작성한 후 저장 및 편집 가능 
- 메모에 사진 추가 가능
- 메모 삭제 가능
